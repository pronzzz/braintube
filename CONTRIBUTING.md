# Contributing to BrainTube

First off, thanks for taking the time to contribute! BrainTube is an open-source project and we love receiving pull requests.

## How Can I Contribute?

### Reporting Bugs

This section guides you through submitting a bug report for BrainTube.

* Use a clear and descriptive title.
* Describe the exact steps to reproduce the problem.
* Include what you expected to happen and what actually happened.

### Suggesting Enhancements

* Use a clear and descriptive title.
* Provide a step-by-step description of the suggested enhancement.
* Explain why this enhancement would be useful.

### Pull Requests

* Follow the Python PEP-8 Style Guide.
* Update `requirements.txt` if needed.
* Add documentation (in `GUIDE.md` or as docstrings) if your PR introduces new logic.
* Ensure all tests pass.

## Development Setup

1. Clone the repo and create your local `venv`.
2. Follow the standard installation in the `README`.
3. Submit a PR when you are ready.
